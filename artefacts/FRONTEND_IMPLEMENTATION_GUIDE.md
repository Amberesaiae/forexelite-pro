## Design System

### Color Palette

```css
--gold: #C9A84C           /* Primary brand color */
--gold-light: #E8C97A     /* Hover states */
--gold-dim: #7A6130       /* Borders, disabled */
--bg-deep: #04080F        /* Page background */
--bg-card: #080E1A        /* Card backgrounds */
--bg-panel: #0C1424       /* Panel backgrounds */
--bg-border: #141E30      /* Borders */
--text-primary: #EEF2FF   /* Main text */
--text-secondary: #8899BB /* Secondary text */
--text-dim: #445577       /* Disabled text */
--green: #00E5A0          /* Buy, profit */
--red: #FF4560            /* Sell, loss */
--blue: #3D85FF           /* Info */
```

### Typography

```css
/* Headings */
font-family: 'Bebas Neue', sans-serif
font-size: clamp(32px, 6vw, 64px)
letter-spacing: 1-3px

/* Body */
font-family: 'DM Sans', sans-serif
font-size: 12-14px
font-weight: 300-700

/* Code/Data */
font-family: 'JetBrains Mono', monospace
font-size: 10-12px
letter-spacing: 1-2px
```

            Dashboard

function NavLink({ href, children, className = '' }: { href: string; children: React.ReactNode; className?: string }) {
  return (
    <Link 
      href={href}
      className={`text-xs font-medium tracking-widest uppercase text-text-secondary hover:text-gold transition-colors ${className}`}
    >
      {children}
    </Link>
  )
}

function MobileNavLink({ href, children, onClick }: { href: string; children: React.ReactNode; onClick: () => void }) {
  return (
    <Link 
      href={href}
      onClick={onClick}
      className="block py-2.5 text-sm font-medium tracking-wide uppercase text-text-secondary border-b border-bg-border hover:text-gold transition-colors"
    >
      {children}
    </Link>
  )
}

## Accessibility Considerations

- All interactive elements have focus states
- Keyboard navigation support (Tab, Enter, Escape)
- ARIA labels for screen readers
- Color contrast ratios meet WCAG AA standards
- Form validation with clear error messages
- Loading states with spinners and text

## Performance Optimizations

- Lazy load chart components
- Virtual scrolling for large tables
- Debounce WebSocket updates (max 10/sec)
- Memoize expensive calculations
- Code splitting by route
- Image optimization (WebP, lazy loading)
